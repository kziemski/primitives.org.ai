Just say what you need.

```typescript
import { ai, list, is } from 'ai-functions'

const qualified = is`${lead} a good fit for enterprise?`
const ideas = list`content topics for ${persona}`
const { summary, nextSteps } = ai`analyze ${salesCall}`
```

## Promise Pipelining

Chain operations without `await`:

```typescript
// Destructure for exactly what you need
const { score, qualified, reason } = ai`qualify ${lead}`

// Chain naturallyâ€”dependencies resolve automatically
const email = ai`follow-up email based on: ${reason}`
const subject = ai`subject line for: ${email}`

// Only await when you need the value
if (await qualified) {
  await send({ subject: await subject, body: await email })
}
```

## Batch with `.map()`

Process arrays in a single call:

```typescript
const leads = list`leads from ${campaign}`

const qualified = await leads.map(lead => ({
  lead,
  score: ai`score 1-100: ${lead}`,
  fit: is`${lead} matches ${icp}?`,
}))
```

## Examples

### Lead Qualification

```typescript
const { score, qualified } = ai`qualify ${lead} for ${product}`

if (await qualified) {
  await assignToSales(lead)
}
```

### Competitive Analysis

```typescript
const { strengths, weaknesses, opportunities, threats } = lists`
  SWOT analysis: ${company} in ${market}
`
```

### Content Pipeline

```typescript
const topics = list`blog ideas for ${audience}`

const evaluated = await topics.map(topic => ({
  topic,
  potential: is`${topic} would drive signups?`,
  outline: ai`outline for: ${topic}`,
}))
```
